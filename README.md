# VeraCryptAssistant
## Назначение:
Программа `Startup` предназначена для консольного монтирования тома VeraCrypt, с последующей автозагрузкой указанных в конфигурационном файле программ.

Путь к `VeraCrypt.exe`, зашифрованным данным, ключевому файлу и сам том для монтирования указываются в конфигурационном файле. Пароль же программа запрашивает при запуске.

Второй проект, программа `Exit`, предназначена для завершения запущенных со смонтированного тома программ с его последующим размонтированием.

Такая связка позволяет быстро и на регулярной основе пользоваться зашифрованным хранилищем VeraCrypt, что повышает безопасность данных и их портативность, в случае хранения зашифрованных данных на внешнем накопителе.

## Примеры использования:
### Startup
Заполнение конфигурационного файла:
```JSON
{
  "MountInfo": {
    "Drive": "I",                                                  // ← Метка для монтируемого тома
    "VeraCryptPath": "E:\\Portable VeraCrypt\\VeraCrypt.exe",      // ← Путь до приложения VeraCrypt
    "DataPath": "S:\\other_files\\CryptedData",                    // ← Путь до зашифрованных данных
    "KeyPath": "K:\\files\\Key"                                    // ← Путь до ключевого файла
  },
  "AppsStartup": [                                                 // ← Список автоматически запускаемых приложений после монтирования
    "I:\\programs\\Portable Punto Switcher\\Punto.exe",
    "I:\\programs\\Portable Telegram\\Telegram.exe",
    "I:\\programs\\Portable Obsidian\\ObsidianPortable.exe",
    "C:\\Users\\Vasyanchik\\AppData\\Roaming\\Resilio Sync\\Resilio Sync.exe"
  ]
}
```

Вызов программы:
```CMD
                    ↓Название программы из $PATH, либо по абсолютному пути
C:\Users\Vasyanchik>st
```

Ввод пароля:
```CMD
Введите пароль для монтирования тома (данные отображаться не будут, для выхода введите значение "q"):
```

Результат:\
После успешного монтирования тома появится соответствующее уведомление в консоли и будут запущены программы, указанные в конфигурационном файле.

### Exit
Заполнение конфигурационного файла:
```JSON
{
  "UnmountInfo": {
    "Drive": "I",                                              // ← Метка размонтируемого тома
    "VeraCryptPath": "E:\\Portable VeraCrypt\\VeraCrypt.exe"   // ← Путь до приложения VeraCrypt
  },
  "AppsExit": [                                                // ← Список автоматически закрываемых приложений перед размонтированием
    "Punto",
    "Telegram",
    "Obsidian",
    "Resilio Sync"
  ]
}
```

Вызов программы:
```CMD
                                 ↓Название программы из $PATH, либо по абсолютному пути
Z:\4_programming\csharp\projects>ex
```

Результат:
```CMD
Процесс "Punto" успешно завершён!
Процесс "Telegram" успешно завершён!
Процесс "Resilio Sync" успешно завершён!
Процесс "Obsidian" успешно завершён!
Том успешно размонтирован!
```
